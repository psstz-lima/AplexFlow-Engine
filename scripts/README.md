# Scripts

Automacoes oficiais do projeto.

- `Run-Mt5Backtest.ps1`: runner base de backtest no `mt5-portable`.
- `Import-CustomCsvHistory.ps1`: importacao de CSV para simbolo customizado do MT5.
- `Search-*.ps1`: buscas amplas por combinacoes de parametros.
- `Refine-*.ps1`: refinamento de presets ja promissores.
- `Run-*.ps1`: execucoes dirigidas e diagnosticos especificos.
- `Finalize-*.ps1` e `Invoke-*.ps1`: consolidacao de rounds e baterias.

Scripts experimentais foram movidos para `scratch/experiments/`.
